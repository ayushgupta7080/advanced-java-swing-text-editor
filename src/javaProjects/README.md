# Advanced Java Swing Text Editor

A feature-rich desktop text editor built using Java Swing.

## Features
- Open and save text files
- Undo and redo functionality
- Find and replace with highlight-all support
- Font customization (family, size, bold, italic)
- Light and dark mode themes
- Recent files menu
- Status bar showing line, column, and character count
- Keyboard shortcuts (Ctrl+F, Ctrl+S, Ctrl+O)

## Technologies Used
- Java
- Swing
- OOP principles
- File I/O
- Git & GitHub

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `StyleEditor.java`
