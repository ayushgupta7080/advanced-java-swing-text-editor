# Advanced Java Swing Text Editor

A feature-rich desktop text editor application built using Java Swing, designed to demonstrate core Java concepts, GUI development, and clean object-oriented design.

This project was developed as part of Java learning and internship preparation, focusing on usability, modularity, and real-world editor features.
